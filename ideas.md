# GLB Model Viewer - 设计思路

## 项目概述
一个专业级的 3D 模型查看器，用于上传、浏览和交互 GLB 格式的 3D 模型。核心特性：左侧结构树 + 右侧 3D 视窗，支持节点选择、高亮与聚焦。

---

## 设计方案选择

### 方案一：工业设计风格（Industrial/CAD-Inspired）
**概率：0.08**

**设计哲学**：模仿专业 CAD 软件（如 Fusion 360、SolidWorks）的审美

**核心原则**：
- 高对比度、深色背景、亮色文本与强调色
- 精确的网格对齐与模块化布局
- 功能优先，视觉简洁但专业

**色彩哲学**：
- 背景：深灰/黑色（#0f1419）
- 文本：浅灰/白色（#e0e0e0）
- 强调色：青蓝色（#00d4ff）或橙色（#ff6b35）
- 逻辑：工业感强，适合工程师使用

**布局范式**：
- 左侧树形结构固定宽度（280px），可拖拽调整
- 右侧 3D 视窗占主要空间
- 顶部工具条紧凑，按钮组织清晰
- 底部信息面板展示选中节点详情

**标志元素**：
- 网格背景或细微纹理
- 锐利的边角，最小圆角（2-4px）
- 单色图标，高对比度

**交互哲学**：
- 快速反馈，无冗余动画
- 鼠标悬停显示工具提示
- 选中状态用高亮色强调

**动画**：
- 最小化动画，快速响应（100-150ms）
- 树节点展开/折叠用简单滑动
- 高亮切换无动画或极短淡入（50ms）

**排版系统**：
- 标题：Roboto Mono Bold（工业感）
- 正文：Inter Regular（易读）
- 层级：通过大小与粗细区分

---

### 方案二：极简主义风格（Minimalism）
**概率：0.07**

**设计哲学**：少即是多，专注内容本身

**核心原则**：
- 最大化空白，减少视觉噪音
- 单一色系 + 一个强调色
- 清晰的信息层级

**色彩哲学**：
- 背景：纯白或浅灰（#fafafa）
- 文本：深灰（#333333）
- 强调色：单一蓝色（#0066cc）
- 逻辑：简洁优雅，适合展示

**布局范式**：
- 对称的两栏布局
- 大量负空间
- 极简工具条（仅必要按钮）

**标志元素**：
- 细线条图标
- 无装饰边框
- 极少圆角

**交互哲学**：
- 隐式反馈，避免弹窗
- 平滑过渡
- 触觉反馈

**动画**：
- 流畅的过渡（200-300ms）
- 树节点展开用缩放 + 淡入

**排版系统**：
- 标题：Playfair Display（优雅）
- 正文：Lato（现代）

---

### 方案三：现代科技风格（Modern Tech/Glassmorphism）
**概率：0.06**

**设计哲学**：未来感、透明感、渐变与模糊

**核心原则**：
- 玻璃态效果（Glassmorphism）与毛玻璃背景
- 渐变与光影营造深度
- 现代感强，视觉冲击力大

**色彩哲学**：
- 背景：深蓝/紫色渐变（#0a1428 → #1a0a2e）
- 文本：白色/浅蓝
- 强调色：霓虹紫/青色（#a855f7 或 #06b6d4）
- 逻辑：科技感强，适合展示

**布局范式**：
- 浮动卡片式布局
- 树形结构用卡片堆叠
- 3D 视窗用毛玻璃边框

**标志元素**：
- 毛玻璃背景（backdrop-filter: blur）
- 渐变边框
- 发光效果

**交互哲学**：
- 丰富的视觉反馈
- 悬停时发光
- 选中时渐变变化

**动画**：
- 流畅的弹簧动画（spring）
- 树节点展开用旋转 + 滑动
- 高亮用发光脉冲

**排版系统**：
- 标题：Space Mono Bold（科技感）
- 正文：Inter（现代）

---

## 最终选择

**选择：方案一（工业设计风格）**

**理由**：
1. GLB 模型查看器是工程/设计工具，用户期望专业感
2. 工业风格提供最高的可用性与信息密度
3. 深色背景对 3D 渲染友好（不分散注意力）
4. 工程师用户熟悉 CAD 软件的交互逻辑

**设计指导**：
- 深灰背景（#0f1419）+ 青蓝强调色（#00d4ff）
- 左侧树形结构固定宽度，可拖拽调整
- 右侧 3D 视窗占主要空间
- 顶部工具条紧凑清晰
- 底部信息面板展示节点详情
- 快速反馈，最小化动画
- 使用 Roboto Mono 强调工业感

## 实现细节

### 色彩系统
- **背景**：#0f1419（深灰黑）
- **卡片**：#1a1f2e（稍浅的深灰）
- **边框**：#2a3142（边界色）
- **文本**：#e0e0e0（浅灰）
- **强调**：#00d4ff（青蓝）
- **成功**：#10b981（绿）
- **警告**：#f59e0b（橙）
- **错误**：#ef4444（红）

### 排版
- **标题**：Roboto Mono Bold, 16-24px
- **正文**：Inter Regular, 12-14px
- **标签**：Inter Medium, 11-12px

### 间距
- 基础单位：4px
- 组件间距：8px, 12px, 16px, 24px, 32px

### 圆角
- 按钮：4px
- 卡片：6px
- 输入框：4px

### 阴影
- 轻：0 1px 2px rgba(0, 0, 0, 0.3)
- 中：0 4px 8px rgba(0, 0, 0, 0.4)
- 重：0 8px 16px rgba(0, 0, 0, 0.5)

### 动画
- 快速反馈：100-150ms
- 过渡：150-200ms
- 缓动：cubic-bezier(0.4, 0, 0.2, 1)
