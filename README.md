# GLB Model Viewer

一个专业级的 3D GLB 模型查看器网页应用，支持模型上传、层级结构浏览、3D 渲染、节点选择与高亮等功能。

## 核心功能

### 1. GLB 上传与加载
- **拖拽上传**：直接拖拽 GLB 文件到上传区域
- **文件选择**：点击上传区域选择本地 GLB 文件
- **进度显示**：加载过程中显示百分比进度条
- **支持大文件**：支持加载 100MB+ 的 GLB 模型
- **Draco 压缩支持**：自动识别并解码 Draco 压缩的 GLB 文件，显著减小文件大小

### 2. 3D 视窗（右侧）
- **WebGL 渲染**：基于 Three.js 的高性能 3D 渲染
- **交互控制**：
  - **旋转**：鼠标左键拖拽旋转模型
  - **平移**：鼠标右键或中键拖拽平移视角
  - **缩放**：鼠标滚轮缩放模型
- **视角控制**：
  - **重置视角**：恢复默认视角
  - **适配模型**：自动调整视角以显示整个模型
  - **聚焦选中**：自动聚焦到选中的节点
- **光照系统**：
  - 环境光（0.6 强度）
  - 主方向光（0.8 强度）
  - 辅助背光（0.3 强度）

### 3. 结构树（左侧）
- **层级展示**：完整的 GLB 场景图层级结构
- **节点类型**：支持 Group、Mesh、Light、Camera 等类型
- **节点信息**：显示节点名称、类型、Mesh 数量
- **展开/折叠**：支持树节点的展开和折叠
- **搜索功能**：支持模糊匹配搜索节点名称
- **节点选择**：点击节点进行选择

### 4. 节点选择与高亮
- **高亮方式**：使用 MeshStandardMaterial 替换材质，应用青蓝色高亮
- **高亮颜色**：#00d4ff（青蓝色）
- **发光效果**：高亮材质具有 0.3 强度的发光效果
- **材质还原**：清除选择后自动恢复原始材质
- **清除选择**：点击"清除选择"按钮或在 3D 视窗空白处清除选择

### 5. 模型统计信息
- **Mesh 数量**：模型中的网格数量
- **三角面数**：总三角面数（精确计算）
- **材质数**：使用的不同材质数量
- **纹理数**：使用的纹理总数
- **灯光数**：场景中的灯光数量

### 6. 节点详细信息
- **节点名称**：显示节点的名称
- **节点类型**：显示节点类型（Group/Mesh/Light/Camera）
- **Mesh 数量**：该节点下的 Mesh 数量
- **三角面数**：该节点下的三角面总数
- **包围盒尺寸**：节点的 X/Y/Z 尺寸
- **中心位置**：节点的中心坐标
- **子节点数**：直接子节点数量

### 7. 导出功能
- **导出选择**：将选中节点的信息导出为 JSON 文件
- **导出内容**：包含节点 ID、名称、类型、路径、Mesh 数量、三角面数等信息

### 核心实现细节

#### GLB 加载流程
1. 用户上传 GLB 文件
2. 使用 FileReader 读取文件为 ArrayBuffer
3. 使用 GLTFLoader 解析 GLB 数据
4. 生成 Three.js Scene 对象
5. 显示加载进度

#### 结构树生成
1. 遍历 Three.js Scene 的所有对象
2. 为每个对象分配唯一 ID（nanoid）
3. 统计 Mesh 数量和三角面数
4. 建立树形结构（parent-child 关系）
5. 在对象上存储 nodeId 便于反向查询

#### 节点选择与高亮
1. 用户点击树节点
2. 查找对应的 Three.js 对象
3. 缓存原始材质
4. 应用高亮材质（MeshStandardMaterial）
5. 自动聚焦到该节点
6. 显示节点详细信息

#### 视角聚焦
1. 计算节点的包围盒
2. 根据包围盒大小计算相机距离
3. 更新相机位置和朝向
4. 更新 OrbitControls 的目标点

## 使用指南

### 基本操作流程

1. **上传模型**
   - 拖拽 GLB 文件到左侧上传区域，或点击选择文件
   - 等待加载完成（显示进度条）

2. **浏览结构**
   - 在左侧结构树中查看模型的层级结构
   - 点击三角形图标展开/折叠节点
   - 使用搜索框快速查找节点

3. **选择节点**
   - 点击结构树中的节点进行选择
   - 选中的节点会在 3D 视窗中高亮显示
   - 视角自动聚焦到选中节点

4. **交互操作**
   - **旋转**：鼠标左键拖拽
   - **平移**：鼠标右键或中键拖拽
   - **缩放**：鼠标滚轮
   - **重置视角**：点击工具条的"重置视角"按钮
   - **适配模型**：点击工具条的"适配模型"按钮

5. **查看信息**
   - 模型统计信息显示在左侧底部
   - 选中节点的详细信息显示在左侧底部
   - 顶部工具条显示当前选中节点的名称和 Mesh 数量

6. **导出选择**
   - 选中节点后，点击"高级选项"展开菜单
   - 点击"导出选择"下载 JSON 文件
   - JSON 文件包含节点的完整信息

### 示例模型

项目包含一个示例模型 `client/public/sample-model.glb`，包含：
- 立方体（Cube）- 青蓝色
- 球体（Sphere）- 橙色
- 圆柱体（Cylinder）- 绿色
- 三个小立方体（SmallCube）- 黄色

## 性能优化

### 已实现的优化
- **材质缓存**：缓存原始材质，避免频繁创建
- **高亮复用**：所有高亮使用同一个 MeshStandardMaterial 实例
- **树节点映射**：使用 ID 快速定位对象，避免全场景遍历
- **OrbitControls**：启用阻尼效果提供平滑的交互体验
- **渲染优化**：启用阴影贴图和反锯齿

### 支持的模型规模
- **小模型**（< 10MB）：即时加载
- **中等模型**（10-100MB）：几秒内加载
- **大模型**（> 100MB）：支持加载但可能需要较长时间

## 已知限制

1. **材质还原**：某些复杂材质（如 PBR 材质）的还原可能不完全
2. **动画支持**：不支持 GLB 中的动画播放
3. **多选**：当前版本不支持多节点同时选择（可在后续版本添加）
4. **拖拽调整**：左侧面板宽度暂不支持拖拽调整（可在后续版本添加）

## 故障排除

### 模型加载失败
- 确保上传的是有效的 GLB 文件
- 检查浏览器控制台的错误信息
- 尝试使用示例模型测试

### 3D 视窗显示异常
- 刷新页面重新加载
- 检查浏览器是否支持 WebGL
- 尝试使用不同的浏览器

### 高亮不显示
- 确保已正确选择节点
- 检查节点是否包含 Mesh
- 尝试调整视角查看

## Draco 压缩支持

本应用支持加载 Draco 压缩的 GLB 文件。Draco 是 Google 开发的 3D 几何压缩库，可以将文件大小减小 80-90%。

**示例**：原始 GLB 60.19 KB → Draco 压缩后 11.37 KB（压缩率 81%）

要生成 Draco 压缩的 GLB 文件，可使用 gltf-transform 工具：

```bash
npm install -g @gltf-transform/cli
gltf-transform draco input.glb output.glb
```
